# 相册访问功能实现计划

## 目录结构
创建 `Sources/Asset` 目录，包含以下文件结构：

```
Sources/
  Asset/
    Core/
      CMAssetManager.swift        # 核心相册管理类
      CMAssetError.swift          # 错误定义
    Models/
      CMAsset.swift               # 图片资源模型
      CMAssetCollection.swift     # 相册模型
    Views/
      CMAssetPickerView.swift     # 相册选择器主视图
      CMAssetPreviewView.swift    # 图片预览视图
      CMAssetGridView.swift       # 图片网格视图
      CMAssetDetailView.swift     # 单个图片编辑视图
    Extensions/
      PHPhotoLibrary+Extension.swift  # Photos框架扩展
      View+Hero.swift             # Hero动画扩展
    CMAssetPicker.swift           # 公开API入口
```

## 核心功能实现

### 1. 权限管理
- 实现相册访问权限请求
- 处理权限状态变化

### 2. 相册管理
- 获取所有相册列表
- 创建新相册
- 删除相册
- 获取指定相册下的图片
- 封装Photos框架的Cursor机制，支持高效访问大量图片

### 3. 图片加载与性能优化
- 使用 `PHImageManager` 高效加载图片
- 实现图片缓存机制
- 使用并发框架处理图片加载
- 利用Photos框架的原生分页能力

### 4. SwiftUI 界面
- **相册选择器主视图**：显示相册列表和图片网格
- **图片网格视图**：支持单选/多选模式切换，显示选择状态
- **图片预览视图**：
  - 支持Hero动画切换
  - 支持图片放大和拖动
  - 原始大小时左右滑动切换图片
  - 向下滑动关闭预览
  - 显示选择状态
- **单个图片编辑视图**：集成项目已有的PhotoEditor功能

### 5. 交互功能
- 单选/多选模式切换
- 图片选择与取消选择
- Hero动画过渡效果
- 图片编辑功能

### 6. API 设计
- 公开API：`CMAssetPicker` 类，提供简单的调用方式
- 内部API：核心实现细节，不对外暴露

### 7. 并发处理
- 使用 `async/await` 和 `Task` 进行异步操作
- 确保主线程更新UI

## 技术要点
- 使用 Photos 框架进行相册操作
- 封装Photos框架的Cursor机制，支持大量图片访问
- 采用 Swift 并发框架处理异步任务
- SwiftUI 实现用户界面
- Hero动画实现流畅过渡
- 集成已有的PhotoEditor功能
- 性能优化确保大量图片时流畅访问
- 清晰的API设计，区分公开和私有接口
